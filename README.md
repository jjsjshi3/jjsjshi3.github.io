# 💖 我们的纪念日 💖

> **“这是专属于情侣们打造的”** > 一个专门为 **各种纪念日** 打造的浪漫计时器

---

## 🚀 浪漫功能

* **⏰ 实时倒数日**：主页超大数字动态计算我们在一起的时间。看着秒数跳动，才发现原来每一秒和你在一起都这么有成就感！
* **🌪️ 弹窗效果**：点一下主卡片，所有的纪念日就像被施了魔法一样“飞”出来。过去的是甜蜜回忆，未来的是美好期待。
* **📝 悄悄话文本**：每张纪念日卡片都有专属的文字（真的！只要写在 `events.js` 里它就一定会出来！）。

---

## 📂 档案柜结构 (Project Structure)

| 档案 | 角色 | 备注 |
| :--- | :--- | :--- |
| `index.html` | **我们的舞台** | 承载了整个浪漫的入口。 |
| `style.css` | **高级化妆师** | 负责磨砂玻璃质感、粉色滤镜和卡片的飞行轨迹。 |
| `script.js` | **中央处理器** | 负责算时间、搞动画，还有盯着浏览器不准偷懒存缓存。 |
| `events.js` | **秘密日记本** | **核心！** 所有的日期和我想对你说的话都存在这里。 |

---

## ✍️ 如何续写我们的故事？ (How to update)

想增加新的纪念日？超级简单！打开 `events.js`，按这个格式填空：

```javascript
{
    title: "我们要去旅行啦 ✈️",
    date: "2026-08-01",
    type: "future",
    text: "想和你去海边看日落，吹吹晚风。"
}
